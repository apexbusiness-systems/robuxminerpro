# Lighthouse Testing

## How to Run Lighthouse

1. Open Chrome DevTools (F12)
2. Go to "Lighthouse" tab
3. Select "Performance", "Best Practices", and "SEO"
4. Click "Generate report"

## Target Scores

- **Performance**: ≥90
- **Best Practices**: ≥90  
- **SEO**: ≥90

## Evidence Required

For each PR, attach:
- Screenshot of Lighthouse scores
- JSON report (click "View report" → gear icon → "Save as JSON")

## Common Issues

- **Performance**: Large images, unused JavaScript
- **Best Practices**: HTTP issues, console errors
- **SEO**: Missing meta tags, poor heading structure

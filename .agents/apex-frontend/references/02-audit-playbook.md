# AUDIT Playbook

## Inputs (use any two)
analytics/funnel, session replays, support tickets, reviews, usability test

## Friction Map Table
Step | Intent | Friction | Severity | Hypothesis | Fix | Metric

## Prioritize
ICE = (Impact × Confidence) / Effort

## Output
- Top 1–3 frictions with evidence
- Proposed redesign deltas (not a full re-skin)
- Verification plan (A/B or pre/post) + guardrails

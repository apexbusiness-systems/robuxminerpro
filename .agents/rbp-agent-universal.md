## EARNING PATHS REFERENCE

### PATH A — Roblox Premium Stipend
- Subscribe to Roblox Premium (450, 1000, or 2200 Robux/month tiers)
- Robux credited automatically each month
- Lowest effort — passive income
- Verify current pricing at: roblox.com/premium/membership

### PATH B — Game Pass (Creator Path)
- Build a game in Roblox Studio
- Create a Game Pass: Roblox Studio → Game Settings → Monetization
- Players purchase the pass once for perks/access in your game
- You earn ~70% of each sale (Roblox takes ~30% platform fee)
- Best practice: Make the game fun WITHOUT the pass — don't gate core gameplay

### PATH C — Developer Products (Creator Path)
- Repeatable purchases inside your game (coins, health, boosts)
- Add via: Roblox Studio → Game Settings → Developer Products
- Great for games with consumable economies
- You earn ~70% per purchase (same fee structure as Game Passes)

### PATH D — Private Servers
- Players pay a monthly fee for a private server in your game
- Set price in Roblox Studio → Game Settings → Private Servers
- Generates recurring monthly income

### PATH E — UGC Marketplace (Advanced)
- Create avatar accessories and items to sell on the Roblox marketplace
- Requires acceptance into the Roblox UGC program
- Apply at: create.roblox.com
- High earning potential for quality items; competitive market

### PATH F — Group Games
- Build games under a Roblox Group
- Revenue flows to Group Funds
- Owner distributes payouts to contributors
- Ideal for team projects and splitting income fairly

### PATH G — Developer Exchange (DevEx) — Cash Out
- Convert your EARNED Robux to real-world money
- Eligibility:
  □ Age 13+ (verified)
  □ Active Roblox Premium subscription
  □ Earned Robux above current minimum threshold
  □ DevEx Terms of Use accepted
- Apply at: roblox.com/devex
- IMPORTANT: Only EARNED Robux convert — purchased Robux do not qualify
- Payout via Tipalti; processing can take several weeks
- Always check the current DevEx rate at roblox.com/devex (rates change)

### PATH H — Clothing Sales
- Design shirts and pants using Roblox templates
- Upload and sell on the marketplace (requires Premium for some items)
- Lower revenue per item but accessible entry point for creators

## SAFETY REFERENCE — RED FLAGS TO WARN USERS ABOUT

```
If user mentions ANY of these → immediately issue a safety warning:

"Free Robux generator"     → Phishing / credential theft scam
"Obby for Robux"           → ToS violation / scam (Roblox doesn't pay for obbies)
"Trading platform"         → Likely ToS violation; account ban risk
"Someone will give Robux"  → Social engineering scam
"Roblox gift card code"    → Gift card scam
"Enter your password"      → Phishing — NEVER share credentials
```

### Safety Warning Template
```
⚠️ SAFETY FLAG: What you're describing sounds like a common Roblox scam.

Legitimate Robux earning NEVER requires:
- Entering your password on a third-party website
- Downloading external software
- Completing "tasks" on unofficial sites
- Sharing your account with anyone

If you've already shared credentials:
1. Change your Roblox password immediately
2. Enable 2-Step Verification at roblox.com/my/account
3. Report to Roblox Support: roblox.com/support
4. Check and revoke unfamiliar authorized apps
```

# A11Y Playbook

## Checks (critical path)
- Role + accessible name for every control
- Focus order matches reading order
- Keyboard works (web/desktop), escape closes modals
- Contrast meets standards; no color-only meaning
- Text scales without truncation; supports large fonts
- Target sizes are usable on touch devices
- Errors are announced and actionable
- Reduced motion supported

## Output
- Findings grouped by severity
- Fixes with verification steps
